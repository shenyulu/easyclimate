Installation Guide#

Welcome to the easyclimate installation guide! üöÄ We‚Äôre excited to help you get started with our powerful climate analysis tool. Follow these simple steps to install easyclimate on your system.

The easyclimate package is currently built and tested for specific platforms due to compatibility and dependency constraints. Below are the supported platforms and notes for users on other systems.

  • Windows x86-64/AMD64 (Windows 10+)

  • Linux x86-64/AMD64 (glibc 2.28 or later, including: Debian 10+, Ubuntu 18.10+, Fedora 29+, CentOS/RHEL 8+)

These platforms are fully tested, and pre-built wheels (.whl) are available on PyPI for easy installation via following methods:

Using the PyPI package manager:

python -m pip install easyclimate

If you don‚Äôt have pip installed, this Python installation guide can guide you through the process.

  1. Install uv

pip install uv

  1. Install easyclimate by uv

uv pip install easyclimate

üõ†Ô∏è Support is coming soon! Stay tuned for updates‚Äîwe‚Äôre working on it!

You can use PyPI to install the latest unreleased version from GitHub (‚ö†Ô∏è NOT recommended in most situations):

python -m pip install --upgrade git+https://github.com/shenyulu/easyclimate@dev

Note

The commands above should be executed in a terminal. On Windows, use the cmd.exe or the ‚ÄúAnaconda Prompt‚Äù app if you‚Äôre using Anaconda.

Tip

For users who have difficulty accessing GitHub, we set up a GIT official mirror to access

python -m pip install --upgrade git+https://gitee.com/shenyulu/easyclimate@dev

Tip

For Linux users who use older systems, special attention is required! It is necessary to install a relatively recent version of the gcc and g++ compiler suites to meet the C++17 requirements. The latest versions of these suites can also be installed as follows:

conda install -c conda-forge gcc gxx

  1. Update package list

sudo apt update

  1. Install the build-essential package by typing the following command:

sudo apt install build-essential

This command installs a bunch of new software packages, including gcc, g++, and make.

  1. You may also need to install the manual pages for development using GNU/Linux:

sudo apt-get install manpages-dev

  1. To verify whether the GCC compiler has been successfully installed, use the following command to print the GCC version:

gcc --version

  1. Start a login shell as the root user.

sudo -i

  1. Install the gcc.

yum install gcc

Press Y and Enter to confirm. Or use the command without confirmation:

yum -y install gcc

  1. Install the g++.

yum install gcc-c++

Press Y and Enter to confirm. Or use the command without confirmation:

yum -y install gcc-c++

Warning

Unfortunately, easyclimate currently does NOT officially support macOS, including both Intel-based Macs and Apple Silicon (M-series) Macs.

Python Version Requirement#

easyclimate requires Python 3.10 or higher. To check your Python version, run:

python --version

Building the Documentation#

Want to build the documentation yourself? üìö Follow these steps:

  • Install the docs build requirements listed above.
    pip install -r docs/requirements.txt

  • Go to the docs directory.

  • Run the build script:
    .\build_docs_windows.ps1

    Hint

    On Windows, we‚Äôve included optipng.exe for you! üòâ You might NOT need to install optipng for image optimization.

    ./build_docs_linux.sh

    Hint

    On Linux, you might need to install optipng for image optimization.

    sudo apt-get install optipng

Tip

For more control, you need to clean the build directory, build the HTML documentation, and copy example notebooks.

About easyclimate-backend#

The easyclimate-backend is the core powerhouse behind the easyclimate front-end package, providing a suite of high-performance, low-level functions for climate data analysis. Implemented in languages like Fortran and C, these functions ensure that your climate data processing is both efficient and accurate.

Because of this, you may also need to install a pre-compiled package or compile it yourself on Windows, Linux, or manylinux package.

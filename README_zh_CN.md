## 👋 总览

**Easy Climate** 是一个Python包，旨在于一行代码分析气候学。


目前支持 Python >= 3.10.

### ✨ 项目目标

* 消除冗长的数据和图形处理代码。
* 使用 numpy 和 xarray 语法，以获得最佳计算速度。
* 使用 Dask 支持可扩展的并行处理，能够处理 TB 或 PB 级别的气候数据。
* 开源软件，提升应用的灵活性。

### 🚀 如何安装

你可以使用 [pip](https://pypi.org/project/pip/) 安装`easyclimate`：

```bash
pip install easyclimate
```

有关安装的更多信息，请查阅文档。

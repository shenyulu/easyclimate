easyclimate.interp.interp_spatial_barnes enables interpolation from site data to grid point data.

See also

  • MeteoSwiss/fast-barnes-py

  • Z√ºrcher, B. K.: Fast approximate Barnes interpolation: illustrated by Python-Numba implementation fast-barnes-py v1.0, Geosci. Model Dev., 16, 1697‚Äì1711, https://doi.org/10.5194/gmd-16-1697-2023, 2023.

meshdata = ecl.interp.interp_spatial_barnes(
    data,
    var_name="qff",
    grid_x=37.5,
    grid_y=75.0,
    point=[-26.0, 34.5],
    resolution=32,
    sigma=1.0,
)
